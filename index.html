Interaktive Webseite

Was ist mit "interkative Webseite" gemeint?

Um eine Webseite interaktiv zu machen, muss alles stimmen im Bezug der Übersichtlichkeit und der Darstellung. Zum einen dürfen auf einer Webseite keine Fehler zu finden sein, zum anderen sollen Formulare korrekt und verständlich dargestellt werden.

Formulare

Ein Formular ist ein wichtiger Anhaltspunkt auf einer Webseite. Formulare sind sehr vielseitig und sollten darum besonderst beachtet werden.

Ein Formular sollte richtig formatiert werden, zudem sollte diese Formatierung passend zum Layout der Webseite sein. Die Reihenfolge der Felder, welche auszufüllen sind, sollte stimmen. Hierfür kann die TAB-Taste zur Testung dienen. Formulare dürfen keine Fehler aufweisen, sonnst wirkt das unprofissionell und hat auf den Besucher sowie das Ranking grossen Einfluss. Ein ganz wichtiger Punkt ist das "responsive Webdesign". Ein Formular muss unbedingt responsive sein, damit die Besucher das Formular einfacher verstehen können und das es ein bessere Position im Netz bekommt (ranking). Das Versenden des Formulars muss einwandfrei funktioneren, sonst kann die Webseite an Position Besuchern leiden. Schlussendlich muss das Formular verständlich und darstellerisch gut sein um eine erfolgreiche Webseite führen zu können.

Was ist ein Skript?

Ein "Script" ist eine Abfolge mehrerer Anweisungen, die ein Computer nacheinander abarbeiten muss. Somit kann dann schlussendlich das Ziel erreicht werden.

Skript schreiben

Um ein Skript schreiben zu können, muss man sich im Vorraus schon über den Vorgang der Befehle Gedanken machen.

    Folgende Schritte müssen beachtet werden:

  1. Ziel festlegen
    Als erstes muss man festlegen, welche Arbeit erledigt werden sollen. Dies muss man sich als Rätsel vorstellen, welches der Computer lösen soll.
  2. Das Skript gestalten
    Um das Skript zu getalten,muss das Ziel in eine Reihe von Aufaben zerlegt werden. Dies kann in Form eines Flussdiagramms ausgeführt werden.
    Anschliessend kann man die Schritte aufschreiben, die der Computer ausführen muss, um die einzelnen Aufgaben zu erledigen. Das ähnelt einem Rezept, das der Computer befolgen kann.
  3. Den Code für die einzelnen Schritte schreiben
    Jeder Schritt muss in einer Programmiersprache geschrieben werden, die der Computer versteht, in unserem Fall in JavaScript.

Skripts entwickeln: Aufgaben

Sobald das Ziel erfasst ist, kann mit den Aufgaben begonnen werden. Hierfür sollte wie schon oben genannt ein Flussdiagramm genutzt werden, um diesen Vorgang klar und übersichtlich darzustellen.

Variablen

Variable erstellen

Eine Variable ist mit der Klassierung in HTML gut zu vergleichen. Bei der Varible folgt als aller Erstes das Schlüsselwort für die Variable selber. Danach folgt der Variablenname, welcher nach eigenen Gunsten bestimmt werden kann.

Beispiel:

var Beispiel;

Um den Wert einer Variable zu definieren, muss der Name des Attributs angegeben werden. Danach folgt der Wert. Sobald der Wert Buchstaben enthält, muss er in "Anführungszeichen" definiert werden. Wie bei jedem anderen Code auch wird hier mit einem ";" abgeschlossen.

Beispiel

var Beispiel;
Beispiel = 3;

Datentypen der Variablen

Man unterscheidet zwischen drei Arten: nummerische-, String- und Boolesche Daten. Der nummerische Datentyp ist für die Zahlen da. Die Strings bestehen aus Buchstaben und anderen Zeichen. Anderst hingegegen können boolesche Daten nur true oder false sein.

//NUMMERISCHE DATEN
//---------------------
0.75

//STRINGDATEN
//---------------------
"Hi, Ivy!"

//BOOLESCHE DATEN
//---------------------
true

Werte der Variablen ändern

Es könnte sein, dass nach einiger Weile die Werte der Attribute geändert werden sollten. Hierfür muss keine komplizierte Lösung gesucht werden, sondern man kann das Attribut nochmals anschreiben und den neuen Wert eingeben.
Sollte es mehrere Werte schon gehabt haben und man möchte nur z.B. den dritten Wert ändern, so muss die Stelle, an welcher sich der Wert befindet, angegeben werden.

var Beispiel_2 = ["Banane","Apfel","Gurke"];
Beispiel_2 (2) = "Mandarine";

Arrays erstellen

Damit kann man nicht nur eine Variable speichern, sondern sogar eine ganze Liste von Werten. Vorallem bei Werten, welche einen Zusammenhang haben, sollte man die Methode mit dem "Array" verwenden.

Operatoren

Grunsätzlich wird zwichen zwei Arten von Ausdrücken unterschieden: "Ausdrücke, die einer Variable einen Wert zuweisen" und "Ausdrücke, die zwei oder mehr Werte verwenden, um einen einzelnen Wert zurückzugeben".

Beispiel

var color = "beige";
var area = 3 * 2;

In Ausdrücken werden Operatoren verwendet, um aus einem oder mehreren Werten einen neuen Wert zu gewinnen. Hier wird zwischen sechs verschiedenen Operatoren unterschieden:

Zuweisungsoperator

Weist einer Variable einen Wert zu.

color = "beige";

Arithmetische Operatoren

Führen die Grundrechenarten durch.

area = 3 * 2;

Stringoperatoren

Verkettet zwei Strings.

greeting = "Hi" + "Molly";

Vergleichsoperatoren

Vergleichen zwei Werte und geben "true" oder "false" zurück.

buy = 3 > 5;

Logische Operatoren

Kombinieren Ausdrücke und geben "true" oder "false" zurück.

buy = (5 > 3) && (2 < 4);

Funktionen

Funktion erstellen

Um eine Funktion zu erstellen, müssen Name und Anweisungen definiert werden. Die Anweisungen, die die Funktion ausführen soll, müssen in geschweiften Klammern sein. Diesen Vorgang nennt man "Funktionsdeklaration".

Beispiel

function sayhello() {
document.write("Hallo!");
}

Funktionen deklarieren, die infos benötigen

Manche Funktionen benötigen Informationen, um ihre Aufgabe zu erfüllen. In einem solchen Fall gibt man der Funktion bei der Deklaration "Parameter" mit. Innerhalb der Funktion werden diese Parameter wie Variablen verwendet.

Beispiel

function getArea(width, height) {
return width * height;
}

Objekte

Was sind Objekte?

Objekte fassen einen Satz von Variablen und Funktionen zu einem Modell zusammen, das für ein Ding der realen Welt steht. In Objekten verwenden wir neue Bezeichnungen für Variablen und Funktionen.

Beispiel

var hotel = {
name: "Quay",
rooms: 40,
booked: 25,
gym: true,
roomTypes: ["Twin","Doppel","Suite"],

checkAvailability: function() {
return this.rooms - this.booked;
}

Objekte erstellen: Literalschreibweise

Die Literalschreibweise ist die einfachste und gängigste Möglichkeit, um Objekte zu erstellen. (Es gibt noch andere Vorgehensweisen.)

Das Objekt bestehet aus den geschweiften Klammern und deren Inhalt. Es wird in der Variable "hotel" gespeichert, sodass Sie als hotel-Objekt darauf verweisen kann. Zwischen den Schlüsseln und ihren Werten steht jeweils ein Doppelpunkt, zwischen dden einzelnen Eigenschaft und Methoden jeweils ein Komma. (Hinter ddem letzten Wert steht kein Komma mehr.)

Beispiel

var hotel = {

name: "Quay",
rooms: 40,
booked: 25,

checkAvailability:function() {
return this.rooms - this.booked;
}

};

If-Anweisungen

If-Anweisungen

Eine if-Anweisung prüft eine Bedingung (oder wertet sie aus). Ist die Bedingung "true", werden die Anweisungen im nachfolgenden Codeblock ausgeführt.

Beispiel

if (scorde >= 50) {
congratulate();
}

Ergibt die Bedingung "true", so wird der nachfolgende Codeblock ausgeführt (also der Codeblock ausgeführt (also der Codeblock zwischen den nächsten geschweiften Klammern).

Ergibt die Bedingung "false", werden die Bedingungen in dem Codeblock nicht ausgeführt. (Das Skript wird am Ende des Codeblocks fortgesetzt.)

If...else-Anweisungen

Auch if...else-Anweisungen prüfen eine Bedingung. Beim Ergebnis "true" wird der erste Codeblock ausgeführt, bei "false"der zweite.

if (score >= 50) {
congratulate();
}
else {
encourage();
}

Switch-Anweisungen

Switch erstellen

Eine switch-Anweisung prüft eine Variable, die als Switch-Wert bezeichnet wird. Jeder der Fälle gibt einen möglichen Wert für die Variable sowie den Code an, der ausgeführt werden muss, wenn sie diesen Wert hat.

switch (level) {

case "One":
title = "Level 1";
break;

case "Two":
title = "Level 2";
break;

case "Three":
title = "Level 3";
break;

default:
title = "Test";
break;

}

Loops

Schleifen Allgemein erklärt

In Schleifen wid eine Bedingung geprüft. Wenn sie "true" ergibt, wird ein Codeblock ausgeführt. Anschliessend wird ide Bedingung erneut geprüft, und wenn sie immer noch true ist, wird der Codeblock erneut ausgeführt. Das wird so lange widerholt, bis die Prüfung "false" zurückgibt. Es gibt drei Arten von Schleifen: "For","While" und "Do...While"

Initialisierung

Die Variable, die als Zähler dient, wird erstellt und auf 0 gesetzt. Gewöhnlich heisst sie "i".

var i = 0;

Bedingung

Die Schleife wird ausgeführt, bis der Zähler einen vorgegebenen Wert erreicht.

i < 10;

Aktualisierung

Jedes Mal, wenn die Anweisungen in den geschweiften Klammern ausgeführt werden, wird der Zähler um 1 erhöht.

i++

For-Schleife

Die For-Schleife braucht man um einen Code in einer bestimmten Anzahl von Durchläufen auszuühren. Dies ist auch die gebräuchlichste Art von der Schleife. Die Bedingung in einer For-Schleife ist ein Zähler, der angibt, wie oft die Schleife durchlaufen werden soll.

Beispiel

for (var i = 0; i < 10; i++) {
document.write(i);
}

While-Schleife

die While-Schleife wird genutzt, wenn man nicht weiss, wie oft der Code ausgeführt werden soll. Die Bedingung muss hier kein Zähler sein. Der Code durchläuft die Schleife so lange, wie die Bedingung "true" ist.

var i = 1;
var msg = "";

while (i < 10) {
msg += i + "x 5 = " + (i * 5) + "<br />";
i++;
}

Do...While-Schleife

Die do...while-Schleife ähnelt der while-Schleife sehr stark, allerdings gibt es einen wichtigen Unterschied: Die Anweisungen in den geschweiften Klammern werden mindestens einmal ausgeführt, auch wenn die Bedingung gleich am Anfang als "false" angezeigt wird.

var i = 1;
var msg = "";

do {
msg += i + " x 5 = " + (i * 5) + "
";s
i++;
} while (i < 1);

Funktionen

In JavaScript ist es wie auch in vielen anderen Programmiersprachen Funktionen zu definieren, welche man später im Skript aufrufen kann. Der Syntax einer solchen Funktion ist folgendermassen aufgebaut:

function fname(Parameterliste){
Anweisungen
};

Der Funktionsname darf nicht mit einer Ziffer beginnen oder Buchstaben wie ä,ö oder ü enthalten. In den Klammern stehen die Parameter, welche mittels Komma getrennt werden, auf die in den Klammern definierten Parameter kann innerhalb der Funktion zugegriffen werden. Eine Funktion kann nicht nur Werte entgegen nehmen sondern auch Werte zurückgeben, ein solcher Wert wird Return Value genannt und wird mittels "return" in der Anweisung der Funktion deklariert.

Beispiel

function quadrat(x) {
return x * x;
}
window.alert(quadrat(5));

jQuery-Syntax

Der jQuery-Syntax ist prinzipiell nach dem Javascript-Syntax aufgebaut. Der jQuery-Syntax hat aber deutlich mehr Vorteile. Er ist kürzer,flexibler und ermöglich effizienteres und übersichtliches Arbeiten.

Syntax

$(selector).action()

Ähnlichkeiten mit CSS

jQuery hat grosse Ähnlichkeiten mit CSS, dabei wirkt die Ansprechung der Klassen und IDs fast identisch.

Beispiel

$(".beispiel-class")
$("#beispiel-id")

#beispiel-id {...}
.beispiel-class {...}

Vorteile gegenüber Javascript

  • es muss weniger Code geschrieben werden
  • Programmierer sparen Zeit bei der Erstellung von JavaScript-Anwendungen
  • das Framework ist einfach zu erlernen
  • jQuery ist standardkonform
  • JavaScript-Anwendungen können stark erweitert werden
  • ein Seitendokument kann schon angezeigt werden, bevor alle Elemente vollständig geladen wurden

Beispiel

var container = document.getElementById('container');


var container = $("#container");
